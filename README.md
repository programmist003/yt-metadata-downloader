# YouTube Metadata Downloader

Простой Python-скрипт для скачивания метаданных YouTube-видео и плейлистов с помощью YouTube Data API v3.

## Что делает

- Получает метаданные видео и плейлистов из YouTube.
- Сохраняет ответ API в JSON-файлы.
- Загружает лучшие доступные миниатюры для каждого ресурса.

## Возможности

- Поддержка URL видео `https://www.youtube.com/watch?v=...`
- Поддержка URL плейлистов `https://www.youtube.com/playlist?list=...`
- Сохранение данных в файлах `video[VIDEO_ID].json`, `playlist[PLAYLIST_ID].json`
- Скачивание миниатюр по URL

## Установка

1. Клонируйте репозиторий или скачайте файлы.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка

В корне проекта должен быть файл `config.toml` со следующим содержимым:

```toml
api_key = "YOUR_YOUTUBE_API_KEY"
```

> Файл `src/auth.py` загружает ключ из `config.toml`, поэтому запускайте скрипт из корневой папки проекта или убедитесь, что текущая рабочая директория содержит этот файл.

## Запуск

Выполните команду из корня проекта:

```bash
python src/main.py URL1 URL2 ...
```

Например:

```bash
python src/main.py https://www.youtube.com/watch?v=dQw4w9WgXcQ https://www.youtube.com/playlist?list=PL... 
```

Если запускаете из папки `src`, используйте:

```bash
python main.py URL1 URL2 ...
```

## Результат

- JSON-файлы с метаданными сохраняются в текущей папке.
- Миниатюры сохраняются в виде файлов `THUMBNAIL_NAME[RESOURCE_ID].ext`.

## Предупреждения

- Убедитесь, что ваш ключ YouTube API включён и имеет доступ к API YouTube Data v3.
- Следите за квотами API: частые запросы могут быстро расходовать лимит.
- Не публикуйте свой API-ключ в публичных репозиториях.

## Структура проекта

- `src/main.py` — основной скрипт загрузки данных
- `src/auth.py` — чтение API-ключа и создание клиента YouTube
- `src/kinds/` — поддерживаемые типы ресурсов: видео и плейлисты
- `src/properties/` — логика получения ID и данных
- `src/utils.py` — вспомогательные функции сохранения и парсинга
