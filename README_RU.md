# YouTube Metadata Downloader

Небольшой инструмент на Python для получения метаданных видео и плейлистов YouTube (YouTube Data API v3).

Основной сценарий (Windows): используйте готовый батник `ytmd.bat` из корня проекта.

Пример использования (Windows — рекомендуется):

```cmd
ytmd.bat < input.txt > responses.json 2> stderr.log
```

Как это работает:
- Ввод читается согласно формату `input.txt` (см. ниже). Вводимые подсказки выводятся в stderr.
- В stdout записывается JSON-массив с сырыми ответами API.

Альтернатива (Linux / macOS / другие платформы): запускайте `src/main.py` напрямую:

```bash
python src/main.py < input.txt > response.json 2> stderr.log
```

Формат `input.txt`:
- Первая строка: пустая (использовать `config.toml`) или ваш ключ API.
- Дальше: URL-адреса видео/плейлистов (по одному на строке).
- Пустая строка отделяет основную секцию от секции плейлистов.
- После пустой строки — URL-адреса плейлистов для получения `playlistItems` (по одному на строке).
- Пустая строка или EOF — окончание ввода.

Примеры и заметки:
- Убедитесь, что `python` доступен в PATH на Windows для запуска `ytmd.bat`.
- `ytmd.bat` — основной пользовательский вход для Windows; `src/main.py` — кроссплатформенный вход.
- Проверка JSON: `python -m json.tool response.json > nul && echo JSON_OK`.
- Не добавляйте `config.toml` с реальным ключом в систему контроля версий.

Важные файлы проекта:
- `ytmd.bat` — батник для Windows (предпочтителен на Windows)
- `run_main.bat` — вспомогательный батник для запуска `src/main.py` с аргументами
- `src/main.py` — основная логика приложения (кроссплатформенный вход)
